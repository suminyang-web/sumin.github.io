# 단일 HTML 도구용 스니펫 모음

새로 단일 HTML 도구를 만들 때 **복붙해서 시작**하기 위한 스니펫 라이브러리.

## 원칙

- 모든 도구는 **단일 HTML 파일**로 배포 (다른 사람에게 파일만 보내면 됨).
- 외부 CDN(PapaParse, XLSX 등)은 OK, 하지만 내가 만든 코드는 inline.
- 빌드 단계 없음. 그냥 복붙.

## 파일

| 파일 | 용도 |
|---|---|
| `csv-utils.js` | CSV/엑셀 데이터 가공에서 자주 쓰는 JS 헬퍼들 (인코딩, 시간 포맷, CSV 다운로드 등) |
| `ui-dropzone.html` | 드래그앤드롭 + 파일 선택 UI 한 덩어리 (HTML + CSS + JS) |

## 사용 흐름

1. 새 도구 HTML 파일을 만든다.
2. 이 폴더의 스니펫에서 필요한 함수/UI를 **복붙**한다.
3. 도구별 로직(필터 조건, 컬럼 매핑 등)을 위에 얹는다.

## 주의

- 스니펫을 고쳐도 **이미 만들어진 도구에는 자동 반영되지 않는다.** 필요하면 각 도구를 손으로 다시 복붙해야 함.
- 두 도구(`report-generator.html`, `수강데이터가공기/index.html`)에서 패턴 추출함 (2026-05-13).
